## ✨ About

**VEN'S ATELIER** is a sophisticated interactive portfolio website designed to showcase creative projects and technical expertise. This platform demonstrates advanced frontend capabilities with smooth animations, custom cursor interactions, and a modern, minimalist aesthetic.

> *You're experiencing it right now!*

## 🚀 Features

### 🎨 **Interactive Design**
- **Custom Cursor System** - Dynamic cursor that adapts to different interactive elements
- **Smooth Animations** - Powered by GSAP and Framer Motion for fluid transitions
- **Responsive Layout** - Optimized for all screen sizes and devices
- **Dark/Light Theme** - Seamless theme switching with smooth transitions

### 🛠️ **Technical Excellence**
- **Modern Stack** - Built with React 19, TypeScript, and Vite
- **Performance Optimized** - Smooth scrolling with Lenis and optimized animations
- **Component Architecture** - Clean, maintainable code structure
- **Type Safety** - Full TypeScript implementation

### 📱 **User Experience**
- **Gallery View** - Beautiful project showcase with hover effects
- **Project Filtering** - Organized by categories (AI, Commerce, Creative Dev, etc.)
- **Detailed Views** - In-depth project information with multiple images
- **Smooth Navigation** - Intuitive user flow with micro-interactions

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hiidonuts/vens-atelier.git
   cd vens-atelier
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173`

### Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run TypeScript type checking
npm run clean    # Clean build artifacts
```
